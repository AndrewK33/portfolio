Вариант использования "Заполнение профиля пользователя, как следующих шаг первой авторизации в приложении"
ID 8
Краткое описание Первый шаг авторизации пользователя в приложении
Актор(ы) "Пользователь", "Система"
Предварительные условия -
Действия 1. ВИ начинается, когда "Пользователь" переходит на экран предлагающий заполнить "Профиль пользователя"
2. "Пользователь" заполняет все предлагаемые поля
3. "Система" проверяет корректность заполненных полей
3.1. В случае наличия несоответствий выводится соответствующее сервисное сообщение
3.2. "Пользователь" корректирует поля
4. После заполнения всех полей "Пользователь" нажимает кнопку "Подтвердить"
5. "Система" автоматически переводит пользователя к "Ленте" приложения
Результат 1. Все поля "Профиля пользователя" заполнены
2. "Пользователь" автоматически направлен к "Ленте" приложения
Ограничения Поле "Никнейм" обязательное для заполнение
Дополнительные атрибуты
 1. Необходимо добавить возможность введения данных в поле "Никнейм" на разных языках
2. Под каждым полем для заполнения необходимо указать ограничения по вводу данных, в случае наличия таковых
