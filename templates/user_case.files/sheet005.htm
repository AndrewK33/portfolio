Вариант использования Возможность выбора страны использования Пользователем
ID 4
Краткое описание Возможность выбора страны использования Пользователем
Актор(ы) "Пользователь", "Система"
Предварительные условия -
Действия 1. ВИ начинается, когда "Пользователь" нажимает на поле "ü"
2. "Система" рыскрывает пролистываемый список стран. Формат списка стран: Строка = "Наименование страны" + "Флаг"
3. "Пользователь" пролистывает список и делает выбор путем нажатия на соответствующую строку
4. "Система" скрывает пролистываемый список стран
5. "Система" меняет язык текста в приложении
Результат Смена отображаемого языка текста в приложении
Ограничения Нельзя выбрать 2 языка использования одновременно
Дополнительные атрибуты
 Корреляция: Флаг страны = Официальный язык/наречие/говор/диалект страны 
Предложения для обсуждения Возможность добавить поле, в котором "Пользователь" может сообщить об отсутствии удобного ему языка
