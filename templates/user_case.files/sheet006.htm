Вариант использования  Возможность ввода номера телефона в соответствии с маской телефона для выбранной страны
ID 5
Краткое описание Ввод номера телефона при регистрации в соответствии с маской телефона для выбранной страны
Актор(ы) "Пользователь", "Система"
Предварительные условия -
Действия 1. ВИ начинается, когда "Пользователь" нажимает на поле "Å" для ввода данных "Номер телефона"
2. "Система" учитывает выбор сделанный в поле "ü" и предлагает готовую маску для ввода данных
3. "Пользователь" вводит данные "Номер телефона"
4. "Система" верифицирует корректность введенных данных
4.1. Поле "Å" подсвечивается зелёным в случае корректности введеных данных.
4.2. Поле "Å" подсвечивается красным в случае некорректности введенных данных
4.2.1. "Пользователь" корректирует введенные данные
4.3. "Пользователь" переходит к заполнению других данных
Результат Поле "Å" заполнено корректно
Ограничения "Пользователь" не может  перейти к заполнению других данных в случае некорректно введенных данных в поле "Å"
Введенные "Пользователем" данные "Номер телефона" не могут совпадать с данными имеющимися в БД "Номера телефонов"
Дополнительные атрибуты
 Необходимо учесть все внутренние правила выбранно страны по набору и длинне номера
